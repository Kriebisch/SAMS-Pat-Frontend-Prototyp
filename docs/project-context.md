# Project Context

## 1. Internship Institution

### 1.1 What is the BIH?

**Berlin Institute of Health (BIH)** focuses on translational medicine:  
converting biomedical research into methods for **prediction, prevention, diagnosis, and treatment**.  
Clinical observations feed back into research, fostering continuous improvement.

### 1.2 Bioinformatics and Translational Genetics

Internship in the **Bioinformatics and Translational Genetics** group, led by **Prof. Dr. Dominik Seelow**.  
Focus: **rare and genetic diseases**, development of software tools for clinicians and researchers.

### 1.3 SAMS & Internship Project

**SAMS (Symptom Annotation Made Simple)** supports:

- HPO – symptom annotation  
- OMIM – genetic disease references  
- Orphanet – rare disease classification  

Data can be stored or exported as **GA4GH Phenopacket**.  
The internship project aimed to develop a **patient-facing frontend** enabling guided symptom documentation.

---

## Internship Goals

- Design a **guided, intuitive interface** for patient-entered data  
- Implement **step-by-step workflows** for onboarding and symptom entry  
- Explore **vanilla web technologies** for interactive UI concepts
